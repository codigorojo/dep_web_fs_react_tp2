The most basic
input types		 Descriptions Examples
text It is used when you want the user to enter a set of characters within an Input.


email Campo para introducir una dirección de correo electrónico.
password Control de linea cuyo valor permanece oculto. Se puede utilizar el atributo maxlength para especificar la longitud maxima del valor a introducir.

tel Control para introducir un número telefónico.Se pueden usar atributos pattern y maxlength para restringir los valores introducidos en este control.


date To enter a date.
time Allows you to enter a time in the form.
datetime-local Control para introducir una fecha y hora (sin zona horaria).
button Brings us a button
submit Needs completing.
image Needs completing.
file Permite al usuario seleccionar un archivo.

Buscar archivo

checkbox Create checkboxes. The value attribute must be used to define the value, the checkbox attribute to indicate the selected element.

Do you have health insurance?
radio Se utiliza el atributo value para definir el valor que se enviará. Se utiliza el atributo checked para indicar si el elemento está seleccionado de forma predeterminada. Los botones radio que tengan el mismo valor para su atributo name están dentro del mismo "grupo" y se puede seleccionar solo 1 a la vez.

input types		 Descriptions Examples
month Permite al usuario seleccionar un mes y año.
reset Botón que restaura los contenidos de un formulario a sus valores predeterminados.


search Needs completing.
range Control para introducir un número cuyo valor exacto no es importante. Este control usa los siguientes valores predeterminados si no se especifica cada atributo
url Campo para editar una URL. El valor introducido se valida para que contenga una cadena vacía o una ruta URL absoluta antes de enviarse. Se puede usar los atributos pattern y maxlength para restringir los valores introducidos en el control.
